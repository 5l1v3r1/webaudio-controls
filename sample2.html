WebAudio-Controls Test Page

WebAudio-Knobs


<webaudio-knob></webaudio-knob>
Default webaudio-knob. If no knob-image is specified, simple embedded image resource is used.



<webaudio-knob src="img/LittlePhatty.png" sprites="100" value="60" min="-100" max="100" step="20" diameter="64"><webaudio-knob>
Knob image file is assigned. Value range can be specified by min/max. The initial value is "60". The value is changed in step of "20".



<webaudio-knob src="img/LittlePhatty.png" sprites="100" diameter="32"><webaudio-knob>
The knob image will be resized to specified size (even recommended to prepare required size image for clear display)



<webaudio-knob src="img/hsw5.png" sprites="4" value="2" min="0" max="4" width="128" height="32"><webaudio-knob>
non-square image is used with specifying width/height instead of diameter.



<webaudio-knob id="knob1" src="img/LittlePhatty.png" sprites="100" diameter="64" valuetip="0" tooltip="Knob Test"></webaudio-knob>
<webaudio-param link="knob1"></webaudio-param>
webaudio-knob can has tooltip-text and editable parameter display field with webaudio-param


WebAudio-Sliders


<webaudio-slider></webaudio-slider>
<webaudio-slider direction="horz"></webaudio-slider>
Default webaudio-slider. If no slider-image is specified, simple embedded image resource is used. The slider direction is specified by "direction" attribute.



<webaudio-slider direction="horz" src="img/hsliderbody.png" knobsrc="img/hsliderknob.png"></webaudi-slider>
Specifying the sliderbase and sliderknob images.



<webaudio-slider id="slider1" direction="horz" src="img/hsliderbody.png" knobsrc="img/hsliderknob.png" min="0" max="1" step="0.1" width="60" height="20" valuetip="0" tooltip="Slider Test"></webaudio-slider>
<webaudio-param link="slider1"></webaudio-param>
Value step is specified, resized, tooltip-text and webaudio-param is added.


WebAudio-Switches


<webaudio-switch></webaudio-switch>
Default webaudio-switch. If no switch-image is specified, simple embedded image resource is used.



<webaudio-switch src="img/switch_toggle.png" width="32" height="32"></webaudio-switch>
<webaudio-switch src="img/switch_toggle.png" width="54" height="54"></webaudio-switch>
Image file is specified. Images are resized to specified size.



<webaudio-switch src="img/switch_toggle.png" type="kick" width="54" height="54"></webaudio-switch>
Kick type switch. In this mode, this switch is on only while pressing.



<webaudio-switch src="img/switch_toggle.png" type="radio" group="radio1" width="54" height="54" value="1" tooltip="Radio-A"></webaudio-switch>
<webaudio-switch src="img/switch_toggle.png" type="radio" group="radio1" width="54" height="54" tooltip="Radio-B"></webaudio-switch&bt;
<webaudio-switch src="img/switch_toggle.png" type="radio" group="radio1" width="54" height="54" tooltip="Radio-C"></webaudio-switch>
Radio type switch. In this mode, only one switch is on in the group.
